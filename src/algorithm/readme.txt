Introduction to the files:
    trainNew.py----------is used to train my new network by transfer learning.
    trainVGG.py----------is used to train original VGG-16 network by transfer learning.
    mylayer.py-----------is the essential custom layer in my new network.
    evaluateNew.py-------is used to test my new network.
    evaluateVGG.py-------is used to test VGG-16 network.
    vggtransfer_last.h5--saved weights and network framework of my new network.
    vggtransfer_VGG.h5---saved weights and network framework of VGG-16 network transferred.
The Running Enviroment:
    python3.6
    keras2.2.4(backend Tensorflow1.12.0)
